一个扩展书签，可以进行书签分组，以及通过分组内的书签切换来跳转代码， 使代码更容易阅读，避免迷失在多层嵌套代码中

### 上手指南

请从本仓库 clone 或者 fork

###### 开发前的配置要求

1. IDEA 2022.2+
2. JDK 11+

###### **安装步骤**

使用 git 克隆仓库到本地

```sh
git clone https://github.com/Nonoas/Bookmark-X.git
```
