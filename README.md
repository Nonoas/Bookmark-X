# Bookmark-X ![](.github/readme/img/logo.svg)

**Bookmark-X** 是一款为 IntelliJ IDEA 打造的书签扩展插件，
允许你按照分组管理代码书签并在组内快速切换，从而在复杂项目中
保持清晰的浏览轨迹。

## 功能特性
- 为文件和代码行添加书签，并按照分组归类管理
- 在书签组之间自由切换，快速跳转到目标位置
- 提供可视化的管理面板，便于查看和编辑

## 快速开始
1. 克隆仓库 `git clone https://github.com/Nonoas/Bookmark-X.git`
2. 在 IntelliJ IDEA 中打开项目并加载 Gradle
3. 运行 `gradle build` 打包插件

更多使用指南请参阅
[项目文档](https://www.nonoas.top/archives/bookmark-x)。
